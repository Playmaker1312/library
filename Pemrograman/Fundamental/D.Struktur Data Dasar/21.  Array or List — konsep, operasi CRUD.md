Mari kita bahas salah satu struktur data paling esensial dalam pemrograman. Jika variabel biasa adalah sebuah kotak kecil untuk menyimpan satu barang, maka materi kali ini adalah bagaimana kita mengelola sebuah lemari arsip berukuran besar.

---

## 1. Definisi Singkat

**Array (atau List)** adalah struktur data yang digunakan untuk menyimpan sekumpulan nilai atau sekumpulan variabel di dalam satu wadah/nama tunggal. Setiap nilai di dalam Array dipisahkan oleh koma dan memiliki nomor urut identitas yang disebut **Indeks (Index)**, sehingga komputer bisa mencari dan memanipulasi data tersebut dengan mudah.

## 2. Analogi: Rak Sepatu Bernomor

Bayangkan Anda memiliki 5 pasang sepatu.

- Daripada meletakkannya berserakan di berbagai sudut kamar (membuat 5 variabel yang berbeda: `sepatu1`, `sepatu2`, dst), Anda membeli sebuah **Rak Sepatu (Array)**.
    
- Di dalam rak sepatu ini, terdapat laci-laci yang berjejer rapi.
    
- Uniknya di dunia pemrograman, penomoran laci ini (Indeks) **selalu dimulai dari angka 0**, bukan 1. Jadi, sepatu pertama Anda taruh di laci nomor 0, sepatu kedua di laci nomor 1, dan seterusnya.
    
- Jika Anda ingin memakai sepatu kets, Anda cukup menyuruh otak Anda: "Ambil sepatu di Rak nomor 2."
    

## 3. Poin-Poin Penting

Dalam mengelola Array/List, _programmer_ selalu berpatokan pada empat operasi dasar yang dikenal dengan singkatan **CRUD**:

- **C - Create (Membuat/Menambah):** Memasukkan elemen baru ke dalam Array (bisa di urutan paling belakang, depan, atau menyisip di tengah).
    
- **R - Read (Membaca):** Mengambil atau melihat nilai dari sebuah elemen berdasarkan nomor urut (Indeks) atau isinya.
    
- **U - Update (Memperbarui):** Mengganti nilai elemen yang sudah ada di nomor urut tertentu dengan nilai yang baru.
    
- **D - Delete (Menghapus):** Menghilangkan elemen dari dalam Array, yang otomatis akan menggeser nomor urut elemen-elemen di sebelahnya.
    

_Catatan:_ Di bahasa modern seperti JavaScript atau Python, satu Array bisa diisi oleh campuran berbagai tipe data (teks, angka, boolean sekaligus), meskipun praktiknya lebih disarankan satu Array berisi satu tipe data yang sama agar rapi.

## 4. Contoh Implementasi

Berikut adalah implementasi **CRUD** pada Array menggunakan **JavaScript**:

JavaScript

```
// [C] CREATE: Membuat Array awal (Laci 0, 1, dan 2)
let keranjangBuah = ["Apel", "Mangga", "Jeruk"];

// Menambahkan buah baru di posisi paling belakang (Laci 3)
keranjangBuah.push("Pisang"); 


// [R] READ: Membaca isi Array
console.log(keranjangBuah[0]); // Output: "Apel" (Membaca laci nomor 0)
console.log(keranjangBuah);    // Output: ["Apel", "Mangga", "Jeruk", "Pisang"]


// [U] UPDATE: Memperbarui isi Array
// Mengganti "Mangga" (yang ada di laci 1) menjadi "Anggur"
keranjangBuah[1] = "Anggur";
console.log(keranjangBuah);    // Output: ["Apel", "Anggur", "Jeruk", "Pisang"]


// [D] DELETE: Menghapus isi Array
// Menghapus 1 buah dari posisi paling belakang ("Pisang" terhapus)
keranjangBuah.pop();
console.log(keranjangBuah);    // Output: ["Apel", "Anggur", "Jeruk"]
```

## 5. Koneksi dengan Web Development

Dalam **Web Development**, Array adalah bentuk data yang akan Anda temui setiap saat, terutama ketika web Anda mulai terhubung dengan _Database_:

1. **Daftar Produk (E-Commerce):** Ketika Anda membuka halaman kategori sepatu di Tokopedia, _database_ mengirimkan data berupa sebuah Array raksasa yang berisi ratusan produk. _Frontend Developer_ kemudian menggunakan perulangan (_Looping_, yang kita bahas sebelumnya) untuk membaca Array tersebut dan menampilkannya menjadi kotak-kotak produk di layar Anda.
    
2. **Keranjang Belanja (Shopping Cart):** Saat Anda mengklik "Tambah ke Keranjang", sistem web memicu operasi **Create (Push)** untuk memasukkan barang tersebut ke dalam Array keranjang Anda. Jika Anda membatalkan barang, sistem memicu **Delete**.
    
3. **Feed Media Sosial:** Beranda Instagram atau X (Twitter) Anda pada dasarnya adalah sebuah Array/List yang berisi objek-objek postingan (_post_). Saat Anda men-_scroll_ ke bawah dan postingan baru muncul, aplikasi sedang memasukkan data baru ke dalam Array tersebut secara dinamis.
