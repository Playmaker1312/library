## 1. Definisi Singkat

**Manipulasi String** adalah proses mengubah, memotong, menggabungkan, atau memformat data teks (_string_) di dalam pemrograman. Karena hampir semua informasi yang dibaca oleh manusia di layar komputer berupa teks, kemampuan untuk meracik teks secara dinamis adalah keterampilan yang sangat krusial.

## 2. Analogi: Bermain Balok Huruf dan Mengisi Formulir

Bayangkan _string_ sebagai **Balok Huruf Mainan** dan **Formulir Kosong**:

- **Concatenation (Penggabungan):** Seperti menyambungkan balok "J-A-M" dengan "T-A-N-G-A-N" agar menjadi kata baru "JAMTANGAN".
    
- **Slicing (Pemotongan):** Seperti mematahkan susunan balok "KEMERDEKAAN" untuk hanya mengambil bagian tengahnya saja, yaitu "MERDEKA".
    
- **Formatting (Pemformatan):** Seperti memiliki formulir cetak bertuliskan _"Halo, nama saya [NAMA], umur saya [UMUR]"_. Anda cukup menempelkan label nama dan umur Anda ke dalam kotak kosong tersebut tanpa harus menulis ulang seluruh kalimatnya dari awal.
    

## 3. Poin-Poin Penting

Ada banyak operasi manipulasi teks, tetapi tiga konsep ini adalah yang paling sering digunakan:

- **Concatenation (`+`):** Menggabungkan dua atau lebih _string_ menjadi satu kesatuan teks yang lebih panjang.
    
- **Formatting / Interpolation:** Memasukkan nilai dari sebuah variabel langsung ke dalam kerangka teks. Di JavaScript modern, ini dilakukan menggunakan _Template Literals_ (menggunakan tanda _backtick_ `` ` `` dan `${...}`). Ini jauh lebih rapi daripada _concatenation_ tradisional.
    
- **Slicing / Substring:** Mengambil sebagian porsi teks berdasarkan nomor urut (Indeks) karakternya. Sama seperti _Array_, huruf pertama dalam _String_ dimulai dari indeks ke-0.
    
- **Case Conversion (Ekstra):** Mengubah seluruh teks menjadi huruf kapital (_Uppercase_) atau huruf kecil (_Lowercase_).
    

## 4. Contoh Implementasi

Mari kita lihat bagaimana **JavaScript** melakukan "sulap" pada teks-teks ini:

JavaScript

```
let namaDepan = "Budi";
let namaBelakang = "Santoso";
let kalimatPanjang = "Belajar Pemrograman Web Sangat Menyenangkan";

// 1. CONCATENATION (Penggabungan dengan tanda +)
let namaLengkap = namaDepan + " " + namaBelakang; 
console.log(namaLengkap); // Output: "Budi Santoso"

// 2. FORMATTING (Lebih rapi dari +)
// Menggunakan backtick (di sebelah kiri tombol angka 1 pada keyboard)
let sapaan = `Halo, ${namaLengkap}! Selamat datang.`;
console.log(sapaan); // Output: "Halo, Budi Santoso! Selamat datang."

// 3. SLICING (Memotong sebagian teks)
// Mengambil karakter dari indeks 0 sampai sebelum indeks 7
let kataPertama = kalimatPanjang.slice(0, 7); 
console.log(kataPertama); // Output: "Belajar"

// 4. CASE CONVERSION (Huruf Besar/Kecil)
console.log(namaDepan.toUpperCase()); // Output: "BUDI"
console.log(namaDepan.toLowerCase()); // Output: "budi"
```

## 5. Koneksi dengan Web Development

Dalam **Web Development**, memanipulasi teks adalah makanan sehari-hari saat mengurus antarmuka pengguna (UI) dan pengolahan data:

1. **Membuat URL Ramah SEO (Slug):** Saat Anda membuat judul artikel web "Belajar JavaScript 101", web akan menggunakan manipulasi teks untuk mengubah teks tersebut menjadi huruf kecil semua dan mengganti spasi dengan tanda strip, sehingga URL-nya menjadi `namablog.com/belajar-javascript-101`.
    
2. **Sistem Pencarian Berdasar Teks (Search):** Jika ada pengguna mencari "SEPATU" dan di _database_ tertulis "sepatu", komputer awalnya akan menganggap itu dua kata yang berbeda. _Programmer_ menggunakan konversi teks (_toLowerCase_) pada kedua kata tersebut sebelum membandingkannya, agar pencarian selalu berhasil terlepas dari penggunaan huruf kapital.
    
3. **Sensor Kata atau Pemendekan Teks (Truncate):** Jika Anda menampilkan cuplikan artikel di halaman utama yang terlalu panjang, Anda menggunakan _slicing_ untuk memotong teks tersebut di huruf ke-100 dan menambahkan `...` di ujungnya agar tampilan _website_ tetap rapi dan tidak kepanjangan.
