## 1. Definisi Singkat

**Operasi Aritmatika** adalah simbol-simbol matematika dasar yang digunakan untuk melakukan perhitungan angka dan menghasilkan nilai baru. **Operator Perbandingan** adalah simbol yang digunakan untuk membandingkan dua buah nilai, di mana hasil akhirnya _selalu_ berupa nilai logika Boolean (**True/Benar** atau **False/Salah**).

## 2. Analogi: Berbelanja di Pasar

Bayangkan Anda sedang berada di kasir pasar swalayan:

- **Operasi Aritmatika** seperti mesin kasir yang menghitung total belanja. (Misal: 3 buah apel **dikali** Rp5.000 **ditambah** 1 buah mangga Rp10.000 = Rp25.000). Hasilnya adalah _sebuah angka baru_.
    
- **Operator Perbandingan** seperti otak Anda yang sedang mengevaluasi uang di dompet. Anda melihat total harga (Rp25.000) dan melihat uang Anda (Rp50.000). Anda bertanya, "Apakah uang saya **lebih besar atau sama dengan** total harga?". Jawabannya hanya bisa dua: "Ya (True)" atau "Tidak (False)".
    

## 3. Poin-Poin Penting

Dalam JavaScript (dan sebagian besar bahasa pemrograman), simbol-simbol yang digunakan sedikit berbeda dengan matematika di sekolah:

**Aritmatika Dasar:**

- `+` (Penjumlahan)
    
- `-` (Pengurangan)
    
- `*` (Perkalian - menggunakan tanda bintang, bukan 'x')
    
- `/` (Pembagian - menggunakan garis miring)
    
- `%` (Modulus - sisa bagi, sangat penting di pemrograman untuk mencari angka genap/ganjil)
    

**Operator Perbandingan:**

- `>` (Lebih besar dari) dan `<` (Lebih kecil dari)
    
- `>=` (Lebih besar atau sama dengan) dan `<=` (Lebih kecil atau sama dengan)
    
- `==` (Sama dengan nilainya)
    
- `===` (Identik - Sama nilainya **dan** sama tipe datanya. Ini paling sering dipakai di JavaScript agar lebih aman)
    
- `!=` (Tidak sama dengan)
    

## 4. Contoh Implementasi

Berikut adalah cara kita menggunakan kedua operator ini dalam JavaScript:

JavaScript

```
// --- OPERASI ARITMATIKA ---
let hargaSepatu = 200000;
let diskon = 50000;

let hargaSetelahDiskon = hargaSepatu - diskon; // Hasil: 150000
let pajak = hargaSetelahDiskon * 0.10;         // Hasil: 15000 (10% pajak)
let totalBayar = hargaSetelahDiskon + pajak;   // Hasil: 165000

console.log("Total Bayar: Rp" + totalBayar);

// --- OPERATOR PERBANDINGAN ---
let uangDompet = 200000;

// Membandingkan total bayar dengan uang di dompet
let isUangCukup = uangDompet >= totalBayar; 
console.log("Apakah uang cukup? " + isUangCukup); // Hasil: true

// Contoh penggunaan identik (===)
let angkaTeks = "5"; // String (Teks)
let angkaBeneran = 5; // Number (Angka)

console.log(angkaTeks == angkaBeneran);  // Hasil: true (Karena nilainya sama-sama 5)
console.log(angkaTeks === angkaBeneran); // Hasil: false (Karena tipe datanya beda, teks vs angka)
```

## 5. Koneksi dengan Web Development

Dalam **Web Development**, kedua operasi ini adalah mesin penggerak fitur-fitur penting:

1. **Aritmatika pada E-Commerce:** Setiap kali Anda melihat keranjang belanja (_shopping cart_) di Tokopedia atau Shopee, operasi aritmatika sedang bekerja keras menghitung (Harga Barang * Kuantitas) + Ongkos Kirim - Potongan Kupon secara _real-time_ saat Anda mengubah jumlah barang.
    
2. **Perbandingan pada Sistem Login/Registrasi:** Saat Anda membuat akun baru, _operator perbandingan_ digunakan untuk memvalidasi data. Misalnya mengecek `umur >= 18` untuk akses konten dewasa, atau mengecek apakah `passwordYangDiketikan === passwordDiDatabase` untuk mengizinkan Anda masuk.
    
3. **Sistem Halaman (Pagination):** Ketika Anda melihat artikel web yang dibagi menjadi beberapa halaman (Page 1, 2, 3), programmer menggunakan **Modulus (`%`)** dan pembagian (`/`) untuk menghitung ada berapa halaman yang dibutuhkan berdasarkan total artikel yang ada di _database_.
