## 1. Definisi Singkat

**Perulangan (Looping)** adalah struktur kontrol yang memerintahkan komputer untuk menjalankan satu blok kode secara berulang-ulang selama kondisi tertentu masih terpenuhi (bernilai _True_). Terdapat dua jenis utama: **`for` loop** (digunakan ketika kita tahu persis berapa kali perulangan harus dilakukan) dan **`while` loop** (digunakan ketika perulangan bergantung pada suatu kondisi yang batas akhirnya belum pasti).

## 2. Analogi: Lari Keliling Lapangan vs Makan Sampai Kenyang

Bayangkan dua aktivitas sehari-hari ini:

- **For Loop (Lari Keliling Lapangan):** Guru olahraga menyuruh Anda, "Lari keliling lapangan 5 putaran!". Anda mulai dari putaran 1, menghitung setiap putaran, dan otomatis berhenti setelah putaran ke-5 selesai. Anda tahu persis kapan harus mulai dan kapan harus berhenti.
    
- **While Loop (Makan Sampai Kenyang):** Anda sedang makan malam. Anda tidak tahu persis berapa suapan yang dibutuhkan untuk kenyang. Aturannya adalah: _Selama (While)_ perut masih lapar, teruslah menyuap makanan. Begitu perut terasa kenyang (kondisi berubah jadi _False_), Anda berhenti makan.
    

## 3. Poin-Poin Penting

- **Struktur `for` Loop:** Terdiri dari tiga bagian dalam satu baris:
    
    1. _Inisialisasi_ (Mulai dari angka berapa, misal: `i = 1`)
        
    2. _Kondisi_ (Sampai kapan harus mengulang, misal: `i <= 5`)
        
    3. _Iterasi/Increment_ (Penambahan setiap putaran, misal: `i++` yang berarti `i` ditambah 1).
        
- **Struktur `while` Loop:** Hanya menerima satu parameter _Kondisi_ di dalam kurungnya. Pengecekan dilakukan sebelum blok kode dieksekusi. Jika sejak awal kondisi sudah _False_, blok kode tidak akan pernah dijalankan sama sekali.
    
- **Bahaya Infinite Loop (Perulangan Tanpa Henti):** Kesalahan paling umum pemula adalah lupa menambahkan instruksi untuk menghentikan loop (misal: lupa mengetik `i++`). Akibatnya, program akan berulang terus-menerus selamanya hingga komputer _hang_ atau kehabisan memori (_crash_).
    

## 4. Contoh Implementasi

Mari kita lihat perbandingan penulisan keduanya dalam **JavaScript** untuk melakukan hal yang sama: mencetak angka 1 sampai 3.

**Menggunakan `for` Loop:**

JavaScript

```
// Sangat ringkas karena semua aturan ada di satu baris
for (let i = 1; i <= 3; i++) {
    console.log("For Loop: Putaran ke-" + i);
}
// Hasil:
// For Loop: Putaran ke-1
// For Loop: Putaran ke-2
// For Loop: Putaran ke-3
```

**Menggunakan `while` Loop:**

JavaScript

```
let putaran = 1; // Inisialisasi dilakukan di luar

// Hanya berisi kondisi "Selama putaran kurang dari atau sama dengan 3"
while (putaran <= 3) {
    console.log("While Loop: Putaran ke-" + putaran);
    
    putaran++; // SANGAT PENTING! Jika ini lupa ditulis, program akan Infinite Loop
}
```

## 5. Koneksi dengan Web Development

Dalam **Web Development**, perulangan sangat krusial untuk mengelola dan menampilkan data (_rendering_):

1. **Menampilkan Daftar Produk/Artikel:** Saat Anda membuka halaman toko _online_, _programmer_ tidak menulis kode HTML satu per satu untuk ratusan produk. Mereka menggunakan `for` loop untuk mengambil daftar produk dari _database_, lalu mengulang desain _kartu produk_ yang sama dan hanya mengganti isinya (gambar, judul, harga) secara otomatis.
    
2. **Membuat Komponen Dinamis (Carousel/Slider):** Fitur galeri gambar yang bisa digeser (_slider_) menggunakan _loop_ untuk memuat semua gambar yang ada dalam sebuah folder sebelum ditampilkan ke layar pengguna.
    
3. **Animasi dan Game di Web:** Jika Anda membuat game berbasis web (menggunakan tag `<canvas>`), Anda akan sering menggunakan _game loop_ berbasis `while` (atau `requestAnimationFrame`) yang terus mengulang pembaruan layar (_render_) selama pemain belum _Game Over_.
