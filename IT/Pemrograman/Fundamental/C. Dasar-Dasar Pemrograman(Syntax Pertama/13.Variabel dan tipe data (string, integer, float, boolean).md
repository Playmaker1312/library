## 1. Definisi Singkat

**Variabel** adalah ruang atau tempat di dalam memori komputer yang digunakan untuk menyimpan data, dan memiliki nama agar mudah dipanggil kembali. **Tipe Data** adalah klasifikasi atau jenis dari data yang disimpan di dalam variabel tersebut, yang menentukan operasi apa saja yang bisa dilakukan pada data itu (misalnya: teks tidak bisa dikalikan dengan teks).

## 2. Analogi: Kotak Penyimpanan dan Isinya

Bayangkan memori komputer Anda seperti sebuah gudang kosong:

- **Variabel** adalah **Kotak Kardus** di dalam gudang tersebut. Anda memberi label pada kotak itu dengan spidol (misal: kotak berlabel "Nama Barang" atau "Harga Barang").
    
- **Tipe Data** adalah **Jenis Barang** yang boleh dimasukkan ke dalam kotak tersebut.
    
    - Anda memasukkan _buku_ ke dalam kotak "Nama Barang" (Teks).
        
    - Anda memasukkan _uang koin_ ke kotak "Harga Barang" (Angka).
        
    - Anda memasukkan _saklar lampu_ ke kotak "Status Tersedia" (Nyala/Mati).
        

Anda tidak bisa menuangkan air (tipe data cair) ke dalam kardus berlubang (tipe wadah yang salah), sama seperti komputer butuh tahu jenis data apa yang sedang ia tangani.

## 3. Poin-Poin Penting

Berikut adalah tipe data paling dasar yang wajib Anda ketahui:

- **String (Teks):** Kumpulan karakter (huruf, angka, simbol) yang diperlakukan sebagai teks. Selalu diapit oleh tanda kutip tunggal (`'...'`) atau ganda (`"..."`). Contoh: `"Budi"`, `"12345"`, `"Halo Dunia!"`.
    
- **Integer (Bilangan Bulat):** Angka utuh tanpa koma/desimal. Bisa positif atau negatif. Contoh: `10`, `-5`, `0`.
    
- **Float / Double (Bilangan Desimal):** Angka yang memiliki nilai pecahan atau koma. Dalam pemrograman, koma biasanya ditulis menggunakan titik (`.`). Contoh: `3.14`, `0.5`, `-99.9`.
    
- **Boolean (Logika):** Tipe data yang hanya memiliki dua kemungkinan nilai mutlak: `true` (Benar) atau `false` (Salah). Digunakan untuk menentukan keputusan (seperti yang kita bahas di materi Logika Boolean sebelumnya).
    

_(Catatan khusus JavaScript: Di JavaScript, Integer dan Float digabung menjadi satu tipe data yang disebut **Number**, namun secara konsep ilmu komputer, keduanya berbeda)._

## 4. Contoh Implementasi

Berikut adalah cara kita membuat variabel (disebut _deklarasi_) dan mengisinya dengan berbagai tipe data menggunakan **JavaScript**:

JavaScript

```
// Menggunakan 'let' untuk membuat variabel yang isinya bisa diubah nanti

// 1. Tipe Data STRING
let namaLengkap = "John Doe";
let pekerjaan = 'Web Developer'; 

// 2. Tipe Data INTEGER (Di JS disebut Number)
let umur = 28;
let jumlahAnak = 0;

// 3. Tipe Data FLOAT (Di JS disebut Number)
let beratBadan = 70.5;
let suhuRuangan = 24.3;

// 4. Tipe Data BOOLEAN
let isSudahMenikah = false;
let isPunyaKTP = true;

// Menampilkan isinya ke layar (console)
console.log("Nama saya " + namaLengkap + ", umur saya " + umur + " tahun.");
```

## 5. Koneksi dengan Web Development

Dalam **Web Development**, variabel dan tipe data adalah bahan bakar yang menggerakkan seluruh data di aplikasi Anda:

1. **String untuk Teks & Input User:** Saat pengguna mengisi formulir pendaftaran, nama, email, dan _password_ yang mereka ketik akan disimpan dalam variabel bertipe _String_ untuk dikirim ke _database_.
    
2. **Number (Integer & Float) untuk E-Commerce:** Saat Anda membuat toko _online_, harga produk disimpan sebagai Integer (misal: `50000`), dan perhitungan diskon pajak menggunakan Float (misal: dikali `0.11`). Anda tidak bisa menggunakan _String_ `"50000"` untuk perhitungan matematika!
    
3. **Boolean untuk Tampilan Web (UI State):** Boolean adalah nyawa dari fitur _toggle_. Contohnya, fitur _Dark Mode_ pada website biasanya dikendalikan oleh variabel seperti `let isDarkMode = true;`. Jika `true`, layar jadi hitam, jika `false`, layar jadi putih.
