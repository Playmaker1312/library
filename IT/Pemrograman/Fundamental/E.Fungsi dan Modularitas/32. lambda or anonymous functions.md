## 1. Definisi Singkat

**Anonymous Function (Fungsi Anonim)** adalah fungsi yang dideklarasikan tanpa memiliki nama pengenal (_identifier_). **Lambda** (sebutan di bahasa Python, Java, C#) atau **Arrow Function** (sebutan di JavaScript) adalah sintaks atau gaya penulisan modern yang sangat ringkas untuk membuat fungsi anonim tersebut. Fungsi ini biasanya dibuat untuk digunakan sekali saja pada saat itu juga (sebagai argumen/parameter untuk fungsi lain).

## 2. Analogi: Pegawai Tetap vs Pekerja Lepas (Freelancer)

Bayangkan Anda sedang membangun rumah:

- **Fungsi Biasa (Named Function):** Ini seperti merekrut **Pegawai Tetap**. Anda memberinya nama panggilan ("Tukang Kayu Budi"), membuat kontrak resmi, dan Anda bisa memanggil "Budi" kapan saja Anda butuh kursi diperbaiki besok, lusa, atau tahun depan.
    
- **Anonymous/Lambda Function:** Ini seperti memanggil **Pekerja Lepas Harian** yang kebetulan lewat di depan rumah. Anda butuh orang untuk mengangkat satu sak semen ke dalam rumah saat itu juga. Anda tidak perlu repot-repot menanyakan namanya, mencatat identitasnya, atau membuat kontrak panjang. Anda hanya memberi instruksi: _"Tolong angkat ini ke dalam, ini upahnya."_ Tugas selesai, dan Anda tidak akan memanggilnya lagi.
    

## 3. Poin-Poin Penting

- **Sintaks Super Ringkas:** Arrow/Lambda function membuang kata kunci seperti `function`, kurung kurawal `{}`, dan bahkan kata `return`. Anda bisa menulis satu fungsi utuh hanya dalam satu baris pendek!
    
- **Dilempar Sebagai Argumen (Callback):** Fungsi anonim paling sering diselipkan langsung ke dalam parameter fungsi lain. Fungsi yang menerima fungsi lain sebagai argumen disebut _Higher-Order Function_.
    
- **Implicit Return (Kembalian Otomatis):** Jika logika di dalam Arrow Function hanya terdiri dari satu baris, komputer akan otomatis mengembalikan (_return_) hasil baris tersebut tanpa Anda harus mengetik kata `return`.
    
- **Perbedaan Istilah:** Konsepnya sama persis, hanya beda nama di tiap bahasa. Python menggunakan kata kunci `lambda`, sedangkan JavaScript menggunakan simbol panah `=>` (sehingga disebut _Arrow Function_).
    

## 4. Contoh Implementasi

Mari kita lihat evolusi dari Fungsi Biasa menjadi Arrow Function di **JavaScript**:

JavaScript

```
// 1. CARA LAMA (Named Function)
// Terlalu panjang untuk tugas sesederhana ini
function kalikanDua(angka) {
    return angka * 2;
}

// 2. CARA TRANSISI (Anonymous Function disimpan di variabel)
const kalikanDuaAnonim = function(angka) {
    return angka * 2;
};

// 3. CARA MODERN (Arrow Function - 1 baris!)
// Kata 'function' hilang, kurung kurawal hilang, kata 'return' hilang
const kalikanDuaModern = (angka) => angka * 2;

console.log(kalikanDuaModern(10)); // Output: 20
```

**Bagaimana contoh pemakaiannya langsung di dalam fungsi lain?** Misalnya kita punya _Array_ dan ingin menjalankan suatu perintah untuk setiap isinya.

JavaScript

```
let daftarHarga = [10000, 20000, 30000];

// Memasukkan Arrow Function ke dalam fungsi bawaan Array .forEach()
daftarHarga.forEach((harga) => console.log("Harganya adalah: Rp" + harga));
// Output:
// Harganya adalah: Rp10000
// Harganya adalah: Rp20000
// Harganya adalah: Rp30000
```

_(Sebagai perbandingan, di **Python** penulisan fungsinya akan terlihat seperti ini: `kalikan_dua = lambda angka: angka * 2`)_

## 5. Koneksi dengan Web Development

Dalam **Web Development** modern, Anda akan melihat _Arrow/Lambda Functions_ di setiap baris kode. Sangat jarang _developer_ modern menggunakan kata `function` utuh kecuali untuk kasus spesifik:

1. **Manipulasi Data (Array Methods):** Saat _Frontend Developer_ mendapat ratusan data produk dari _database_ dan ingin menyaring (_filter_) produk yang harganya di atas Rp50.000, mereka akan menggunakan Arrow Function: `produk.filter(item => item.harga > 50000)`. Sangat elegan dan mudah dibaca.
    
2. **Event Listener (Klik Tombol):** Saat mengatur apa yang terjadi jika tombol diklik, kita tidak perlu membuat fungsi bernama khusus. Kita cukup menempelkan fungsi anonim: `tombol.addEventListener('click', () => alert('Tombol diklik!'))`.
    
3. **Proses Asinkron (Fetching API):** Saat mengambil data dari internet (yang memakan waktu), kode ditulis menggunakan fungsi anonim untuk menangani data yang masuk nanti: `fetch('url').then(data => data.json())`.
