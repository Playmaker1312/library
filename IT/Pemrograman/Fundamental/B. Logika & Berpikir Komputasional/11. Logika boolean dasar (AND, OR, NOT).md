
## 1. Definisi Singkat

**Logika Boolean** adalah sistem logika matematika yang hanya memiliki dua nilai: **Benar (_True/1_)** atau **Salah (_False/0_)**. Dalam pemrograman, logika ini digunakan untuk mengevaluasi kondisi dan membuat keputusan menggunakan tiga operator utama: **AND** (Dan), **OR** (Atau), dan **NOT** (Bukan/Kebalikan).

## 2. Analogi: Syarat Masuk Bioskop

Bayangkan Anda sedang bekerja sebagai penjaga pintu bioskop untuk film dewasa (18+):

- **AND (Syarat Ketat):** Pengunjung boleh masuk JIKA mereka "Membawa Tiket" **AND** "Berusia 18+". Jika mereka bawa tiket tapi masih 15 tahun? Ditolak. Jika 20 tahun tapi tidak bawa tiket? Ditolak. _Keduanya harus terpenuhi (Benar/True)._
    
- **OR (Syarat Fleksibel):** Untuk membayar, pengunjung boleh menggunakan "Uang Tunai" **OR** "Kartu Debit". Punya salah satu saja sudah cukup untuk bisa membayar. _Hanya butuh minimal satu yang terpenuhi (Benar/True)._
    
- **NOT (Pembalikan):** Ada aturan bahwa karyawan dilarang menonton saat jam kerja. Jadi, pengunjung boleh masuk JIKA mereka **NOT** "Karyawan Bioskop". Jika dia karyawan (True), maka dibalik menjadi dilarang (False).

## 3. Poin-Poin Penting

Dalam JavaScript (dan sebagian besar bahasa pemrograman lainnya), simbol operator ini ditulis secara khusus:

- **AND (`&&`):** Hanya menghasilkan nilai `true` jika **semua** kondisi di sebelah kiri dan kanannya bernilai `true`.
    
- **OR (`||`):** Menghasilkan nilai `true` jika **salah satu atau semua** kondisi bernilai `true`. Hanya bernilai `false` jika semuanya `false`.
    
- **NOT (`!`):** Membalikkan keadaan. Jika kondisinya `true`, diubah menjadi `false`. Jika `false`, diubah menjadi `true`.
    
- **Kombinasi:** Anda bisa menggabungkan ketiganya menggunakan tanda kurung `()` layaknya operasi matematika untuk membuat aturan yang sangat kompleks.
    

## 4. Contoh Implementasi

Berikut adalah bagaimana kita menulis logika penjaga pintu bioskop tadi menggunakan JavaScript:

JavaScript

```
let bawaTiket = true;
let usia = 20;
let isKaryawan = false;

// 1. Contoh AND (&&)
if (bawaTiket === true && usia >= 18) {
    console.log("Silakan masuk studio!"); // Ini akan berjalan
} else {
    console.log("Maaf, Anda tidak memenuhi syarat.");
}

// 2. Contoh OR (||)
let bayarCash = false;
let bayarDebit = true;

if (bayarCash === true || bayarDebit === true) {
    console.log("Pembayaran berhasil!"); // Ini akan berjalan karena bawa debit
}

// 3. Contoh NOT (!)
if (!isKaryawan) {
    console.log("Anda adalah pengunjung biasa, selamat menonton."); // Berjalan karena isKaryawan aslinya false, dibalik (!) menjadi true
}
```

## 5. Koneksi dengan Web Development

Logika Boolean adalah nyawa dari fitur interaktif di **Web Development**:

1. **Validasi Formulir (Form Validation):** Saat Anda membuat halaman _Sign Up_, JavaScript akan menggunakan operator `&&` untuk mengecek: JIKA "Email valid" **AND** "Password lebih dari 8 karakter" **AND** "Password cocok", maka tombol _Daftar_ baru bisa diklik.
    
2. **Sistem Filter Pencarian:** Saat pengguna mencari produk di e-commerce, web menggunakan operator `&&` dan `||`. Misalnya: Tampilkan produk yang ("Kategori Sepatu" **AND** "Warna Hitam") **OR** ("Diskon di atas 50%").
    
3. **Hak Akses (Authorization):** Di sisi _Frontend_ maupun _Backend_, logika `NOT (!)` sering digunakan untuk melindungi halaman. Misalnya: `if (!isLoggedIn)` (Jika pengguna BELUM login), maka tendang mereka kembali ke halaman _Login_.
